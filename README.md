# Dendrometer Empetrum hermaphroditum

### Various analysis for dendrometer measurements from Empetrum hermaphroditum specimens

# <span style="color:green"> **About** </span> 

This Repository contains various scripts to prepare, analyse and plot dendrometer measuremens from Empetrum hermaphroditum specimens in Central Norway. The individuals were specifically selected because they showed a strong stem shrinking in some years or continiously in some cases.

# <span style="color:green"> **Install dependencies** </span> 

In a bash console run:

`python -m venv .venv` or `python3 -m venv .venv` (depending in your version)
`source .venv/bin/activate`  On Windows: `.venv\Scripts\activate`
`pip install -r requirements.txt`
